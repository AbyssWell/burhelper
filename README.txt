БурХелпер - небольшой каталог программ для облегчения жизни бурильщика.

Основные преимущества:
- работает на чистом html и javascript > не требует установки
- малый вес, если загружается на устройство
- может работать из памяти телефона без интернета
- не запрашивает и не собирает геолокацию и личные данные
- открытый код с возможностью просмотра всех изменений в коде и их описания
- не работает в фоне на мобильных устройствах

"Я хочу пользоваться программой без интернета. Как это сделать?"
1. На главной странице БурХелпер нажмите "Скачать для оффлайн использования". Скачается архив "burhelper-main.zip".
2. В зависимости от вашей платформы сделайте следующее:
    - Android:
        Единоразовая настройка:
        1. Откройте скачанный архив "burhelper-main" в папке "Downloads" и распакуйте в любое удобное для вас место. Например в ту же папку.
        2. Скачайте приложение "Simple HTTP Server" из Google Play. Это приложение создает локальный сервер на вашем смартфоне и необходимо для запуска БурХелпер в оффлайн режиме.
            2.1 Откройте приложение. Проверьте, чтобы были установлены галочки на первых двух пунктах "Redirect..." и "Render...". Если они отсутствуют - установите их.
            2.2 В пункте "Root Folder" нажмите на иконку папки, выберите "Storage Access Framework", зайдите в папку "burhelper-main" и нажмите "Использовать эту папку". В пункте "Root Folder" теперь должно отображаться "burhelper-main".
        Первый и последующие запуски БурХелпер:
        1. Откройте "Simple HTTP Server", нажмите Start.
        2. В пункте "IP" нажмите на "127.0.0.1". Выберите браузер, через который открыть БурХелпер.

    - PC / персональный компьютер:
        1. Откройте скачанный архив через проводник или программу (например WinRaR или 7-zip). Переместите папку "burhelper-main" из архива в любое удобное для вас место.
        2. В папке "burhelper-main" откройте файл "index.html"
