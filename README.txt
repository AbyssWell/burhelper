БурХелпер - небольшой каталог программ для облегчения жизни бурильщика.

Основные преимущества:
- работает на чистом html и javascript > не требует установки
- малый вес, если загружается на устройство
- может работать из памяти телефона без интернета
- не запрашивает и не собирает геолокацию и личные данные
- открытый код с возможностью просмотра всех изменений в коде и их описания
- не работает в фоне на мобильных устройствах

"Я хочу пользоваться программой без интернета. Как это сделать?"
1. На главной странице БурХелпер нажмите "Скачать для оффлайн использования". Скачается архив "burhelper-main.zip".
2. В зависимости от вашей платформы сделайте следующее:
    - Android:
        1. Откройте файловый менеджер на смартфоне и найдите папку "Downloads", в ней найдите скачанный ранее архив. Вы можете использовать любой файловый менеджер, поддерживающий работу с zip-архивами.
        2. Откройте скачанный архив и распакуйте в любое удобное для вас место. Например в ту же папку.
        3. Откройте распакованную папку "burhelper-main" и найдите файл "index.html".
        4. Вариант 1: Нажмите на файл, файловый менеджер попросит выбрать приложение для открытия файла. Выберите любой браузер (Mozilla, Chrome или др), установленный на смартфоне.
           Вариант 2: Долгое нажатие на файл > пункт "Открыть с помощью". Выберите любой браузер (Mozilla, Chrome или др), установленный на смартфоне.
        5. Готово! Можно пользоваться программой. Для последующего удобства можете добавить страницу в закладки браузера.
    - PC / персональный компьютер:
        1. Откройте скачанный архив через проводник или программу (например WinRaR или 7-zip). Переместите папку "burhelper-main" из архива в любое удобное для вас место.
        2. В папке "burhelper-main" откройте файл "index.html"
        3. Готово! Можно пользоваться программой. Для последующего удобства можете добавить страницу в закладки браузера.
P.S. Если вы создали закладку в браузере не перемещайте никуда папку "burhelper-main", иначе браузер не сможет открыть страницу через закладку. Выбирайте место хранения программы с умом.
